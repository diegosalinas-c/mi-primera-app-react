# Mi Primera Aplicación en React

## Descripción
Este proyecto corresponde a una aplicación simple de lista de tareas desarrollada con React. La aplicación permite al usuario escribir tareas, agregarlas a una lista y eliminarlas cuando ya no sean necesarias.

## Objetivo
El objetivo de esta aplicación es aplicar conceptos básicos de React, especialmente el uso de componentes, el manejo del estado con useState y la interactividad con el usuario.

## Tecnologías utilizadas
- React
- JavaScript
- HTML
- CSS
- Vite

## Estructura del proyecto
- App.jsx: componente principal
- Header.jsx: encabezado de la aplicación
- TaskForm.jsx: formulario para agregar tareas
- TaskList.jsx: listado de tareas
- TaskItem.jsx: tarea individual
- index.css: estilos personalizados

## Funcionalidades
- Agregar tareas
- Mostrar tareas en pantalla
- Eliminar tareas
- Actualización dinámica sin recargar la página

## Instalación y ejecución
1. Instalar dependencias:
   npm install

2. Ejecutar el proyecto:
   npm run dev

## Uso
El usuario debe escribir una tarea en el campo de texto y luego presionar el botón "Agregar". La tarea aparecerá en pantalla automáticamente. Para eliminar una tarea, debe presionar el botón "Eliminar".

## Decisiones de diseño
Se utilizó una interfaz simple, clara y ordenada, con colores suaves y estructura modular para facilitar la comprensión del código y mejorar la experiencia de uso.
